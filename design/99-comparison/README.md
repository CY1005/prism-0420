# 设计 vs Prism 现状对照报告

> 模块设计完成后，在此输出对照报告。

---

## 对照颗粒度

- **实体级**：实体存在性、字段差异
- **状态机级**：状态枚举、转换合法性
- **数据流级**：流转路径、转换逻辑
- **接口级**：API 命名、参数、响应

---

## 差异判定标准（CY 已认可）

| 差异类型 | 处理方式 |
|---------|---------|
| 数据模型字段缺失/类型错误 | **必须重构** |
| 状态机非法转换/遗漏状态 | **必须重构** |
| 缺少事务边界 | **必须重构**（数据一致性问题） |
| 分层不清晰 | 看影响面，不一定立刻动 |
| 字段命名不规范 | 小改，不算重构 |
| 重复代码 | 不算（这是另一回事） |
